# SaaS画面録画ガイド — X釣り動画用

threads-auto-post / note-auto / mytrack など既存SaaSを「自動で動いてる感」で見せるための実践ガイド。

## なぜこれが一番釣れるか

- **本物の画面**だから信憑性が桁違い (Remotionは作り物感が出る)
- 量産可能 (1本のレコーディングから複数の切り抜きが作れる)
- ストック化できる (撮り溜め → 数週間〜数ヶ月分の素材)

---

## 録画ツール選定

### Windows推奨スタック

| ツール | 用途 | 価格 |
|---|---|---|
| **OBS Studio** | メイン録画 (高品質 + 多ソース合成) | 無料 |
| **ShareX** | 短いクリップ録画 + GIF化 | 無料 |
| **Windows Game Bar** | 緊急時の即録画 (Win+G) | 無料 |
| **CapCut Desktop** | 編集 (倍速・テロップ・BGM) | 無料 |
| **DaVinci Resolve** | 本格編集 | 無料 |

### OBS推奨設定 (X用)

```
出力解像度: 1080×1920 (縦9:16)
FPS: 60
ビットレート: 8000 Kbps
形式: MP4 (H.264)
音声: AAC 192 Kbps
```

縦動画にするため、OBSのキャンバスを縦に設定 → ブラウザを縦長ウィンドウにする。

## 撮影スクリプト テンプレ

### A) "勝手に投稿されてる"系 (threads-auto-post)

**長さ目標:** 15-30秒
**構成:**

1. **0-2秒:** タイトルテロップ被せ
   - 「寝てる間にXに自動投稿させたら…」
   - 「Claude Codeで作った投稿Botの管理画面」

2. **2-8秒:** 管理画面ダッシュボード
   - 投稿予定リスト (ずらっと並んだ未来の投稿)
   - 「次の投稿まで: 24分」みたいなカウントダウン
   - **マウスはゆっくり動かす** (素早すぎると認知されない)

3. **8-15秒:** 投稿が公開された瞬間
   - 別ウィンドウでX投稿が表示される
   - インプレッション数が増えていく (時間倍速で見せてもOK)

4. **15-25秒:** 結果
   - 「今日の自動投稿: ○件」
   - 「フォロワー: +○○」
   - 売上があれば「今月: ¥○○○」

5. **25-30秒:** オチ / CTA
   - 「作業時間: 0時間」
   - 「リプで配布」「プロフから」

### B) "並列エージェント動いてる"系

**長さ目標:** 10-20秒
**構成:**

1. ターミナル分割画面 (4分割または6分割) — Windows Terminal の Panes
2. 各ペインで別のスクリプトが流れる
   - ペイン1: `pnpm dev` のログ
   - ペイン2: API呼び出しログ
   - ペイン3: DB クエリ
   - ペイン4: テスト実行
3. 倍速にすると「同時にめっちゃ動いてる」感が出る

**準備コマンド (Windows Terminal):**
```powershell
# 4分割ペイン起動
wt -p "Windows PowerShell" `; split-pane -p "PowerShell" `; split-pane -H -p "PowerShell" `; move-focus left `; split-pane -H -p "PowerShell"
```

### C) "数字が伸びていく"系

**長さ目標:** 8-15秒
**構成:**

1. mytrack / note-auto / threads-auto-post のダッシュボード
2. ブラウザDevToolsで数字を一時的に書き換え (魅せ用)
   - `document.querySelector('[data-stat]').textContent = '1,247'`
3. もしくはRefreshで実際の更新を見せる
4. 1秒に1回ぐらいの間隔で数字が増えるテンポが理想

## 撮影前チェックリスト

- [ ] **個人情報を隠す** — 本物のメールアドレス・トークンが映ってないか
- [ ] **顧客データを隠す** — souken / juku-report は要注意
- [ ] **ブラウザのブックマーク・タブを整理** — 余計な情報が映らないように
- [ ] **通知をOFF** — Discord, Slack, メール
- [ ] **デスクトップを綺麗に** — 関係ないファイル隠す
- [ ] **ダークモード推奨** — X上で目を引く + 高級感
- [ ] **マウスカーソルを大きく** — Windowsの「マウス設定 > サイズ」

## 撮影後のCapCutワークフロー

1. **インポート** → 30秒以下にカット
2. **倍速** 1.5x〜2x が映え + 維持率UP
3. **テロップ** 大きいフォント (60-80pt)、黒バック白文字 or 白バック黒文字
4. **効果音** クリック音・成功音・通知音をかぶせる
   - CapCutの「サウンド」→「効果音」→「テック」検索
5. **BGM** ローファイ or トラップ系 (CapCut無料音源で十分)
6. **テンポ** 0.5秒に1カット切り替え目安
7. **エクスポート** 1080×1920 / 60fps / 高画質

## ストック運用

1本撮ったら：
- フル尺 (30秒)
- 切り抜き1 (ダッシュボード部分のみ・10秒)
- 切り抜き2 (投稿の瞬間のみ・8秒)
- 切り抜き3 (数字伸びる部分のみ・10秒)

→ **1本の素材から4-6投稿**作れる。これを既存SaaS全部 (threads-auto-post / note-auto / mytrack / invoice-saas2) でやれば、数週間分の素材になる。
